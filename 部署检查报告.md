1. 登录 https://console.cloud.tencent.com/cam/capi 获取 SecretId/SecretKey
2. 执行：
   ```bash
   tcb login --apiKeyId <SecretId> --apiKey <SecretKey>
   tcb fn deploy login -e cloud1-d8gshwhb5dc72723e --force
   tcb fn deploy habits -e cloud1-d8gshwhb5dc72723e --force
   tcb fn deploy checkin -e cloud1-d8gshwhb5dc72723e --force
   tcb fn deploy stats -e cloud1-d8gshwhb5dc72723e --force
   ```
